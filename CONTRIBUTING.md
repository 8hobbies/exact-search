# Contributing

## Build and Run

To build, run

```bash
npm install
npm run build
```

The extension is built in the `dist/` directory.

## Lint

Run

```bash
npm run lint
```

## Homepage Content

To make changes to the content of the home page, please edit the files in the folder descriptions/.
